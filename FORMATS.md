# Formats

## Supported (Export)

**EDL (Edit Decision List)** - Text, basic cuts only
- CMX 3600: Standard (Premiere, Resolve, Media Composer)
- OpenShot: Gaps, unified audio (OpenShot, Kdenlive)
- GVG: 6-char reels (Grass Valley)
- CMX 340: 3-char numeric reels, 2 audio max (Legacy)

## Planned

**MLT XML** - Shotcut/Kdenlive, 10-15h
- XML, effects, transitions
- Shotcut, Kdenlive, Flowblade
- https://www.mltframework.org/docs/mltxml/

**FCP7 XML** - Industry standard, 15-25h
- Premiere, Resolve, Media Composer, FCP7
- Nested sequences, effects, transitions

**FCPXML** - Final Cut Pro X only, 20-30h
- FCPX, Resolve (limited)

**AAF** - Not planned (binary, 40-60h, needs library)
